# An Simple Implementation of Blockchain in Golang
## Usage
```sh
go run main.go
```
and then access:
- to mine a new block
```
localhost:8080/mine
```
- to initiate a new transaxtion
```
localhost:8080/transaction/new
```
- to get the current full chain
```
localhost:8080/chain
```
